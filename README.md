# 차이점 찾기 게임 🎯

> Vercel 배포 완료 - 2025년 1월 업데이트 (v2.0)



## 📖 프로젝트 소개
Supabase를 활용한 현대적인 틀린그림찾기 게임입니다.

## 🎮 주요 기능
- **게임 선택**: 다양한 난이도별 이미지 선택
- **게임 플레이**: 실시간 틀린곳 찾기
- **결과 확인**: 점수 및 성과 표시
- **이미지 비교**: 정답 확인 모드

## 🗂️ 파일 구조
