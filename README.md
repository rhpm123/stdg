# 🎯 틀린그림찾기 게임

Korean Spot-the-Difference Web Game

## 🚀 실시간 플레이

- **웹사이트**: [여기서 플레이하세요!](https://your-game-url.vercel.app)
- **관리자 페이지**: [/admin](https://your-game-url.vercel.app/admin)

## 📁 프로젝트 구조

```
├── index.html          # 메인 페이지
├── game-select.html    # 게임 선택 페이지  
├── game-play.html      # 게임 플레이 페이지
├── game-result.html    # 게임 결과 페이지
├── compare-images.html # 관리자 페이지
├── vercel.json         # Vercel 배포 설정
└── assets/
    ├── css/            # 스타일 파일들
    └── js/             # JavaScript 게임 로직
```

## 🎮 게임 기능

- **실시간 틀린그림찾기**
- **좌표 기반 정답 검증**
- **점수 시스템**
- **하트 시스템**
- **Supabase 백엔드 연동**

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Supabase (PostgreSQL, Storage, Edge Functions)
- **Deployment**: Vercel
- **Database**: PostgreSQL (Supabase)

## 📦 배포

이 프로젝트는 **Vercel**에 자동 배포됩니다.

1. GitHub에 푸시
2. Vercel이 자동으로 빌드 및 배포
3. URL을 통해 즉시 접근 가능
